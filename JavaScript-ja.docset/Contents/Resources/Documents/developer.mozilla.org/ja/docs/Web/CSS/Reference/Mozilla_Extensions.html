    Mozilla はベンダー接頭辞（ -moz- ）付きの CSS 独自拡張を多くサポートしています。

    これらのプロパティ内の幾つかは標準仕様の最終勧告に向けて策定中のものです。既に CSS の仕様書の草案には記載されていますが、まだ実験段階であり、最終的に標準となるプロパティは現在のベンダー接頭辞付きプロパティの実装とは異なる場合があります。これら非標準のプロパティの一部は、XUL 要素にのみ適用されます。

    一旦標準化され、接頭辞無しの形式がサポートされ次第、ベンダー接頭辞付きのプロパティは順次削除されます。
