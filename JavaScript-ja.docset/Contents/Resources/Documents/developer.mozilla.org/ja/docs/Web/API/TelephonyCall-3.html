    Non-standard
    This feature is not on a current W3C standards track, but it is supported on the Firefox OS platform. Although implementations may change in the future and it is not supported widely across browsers, it is suitable for use in code dedicated to Firefox OS apps.

    This API is available on Firefox OS for certified applications only.

    Summary

    A reference to a function to be called to handle the Telephony API's alerting event. The event handler is called with a CallEvent as its only parameter.

    Note: When this event occurs?

    Syntax

    instanceOfTelephonyCall.onalerting = funcRef;

    Where funcRef is a function to be called when the alerting event occurs. These events are of type CallEvent.

    Specifications

    This is a non standard implementation, but it is discussed at W3C as part of the System Application Working Group.
