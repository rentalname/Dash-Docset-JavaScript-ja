    Mozilla における ECMAScript 6 のサポート

    In This Article
    1. 既に Firefox でサポートされている機能
      1. 構文
      2. 標準ライブラリ
    2. まだ Firefox でサポートされていない機能
    3. 関連情報

    ECMAScript 6 は "Harmony" または "ES.next" のコードネームで呼称される、JavaScript の次期標準仕様です。策定中の仕様草案は公式の wiki で閲覧可能です。es-discuss よりフィードバックを提供する事が出来ます。

    TC39 Consensus は 3 段階のレベルに位置づけられています:

    Consensus Ongoing Discussion Controversial

    これらは機能やデザインについて、TC39 でのおおまかな合意状況を表します。3 つのレベルの意味は以下のとおりです:

    • Consensus: デザインに関して、重大な意見の相違は残っていません。
    • Ongoing discussion: 機能について意見が一致していますが、デザインについて議論が続いている状況です。
    • Controversial: デザインに関して重要な議論が続いています。
