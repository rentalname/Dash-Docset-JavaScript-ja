    HTML5 Demos

    最新の HTML 技術を用いたデモのコレクション

    HyperText Markup Language (HTML) は、Web ページの製作や、その他のブラウザで閲覧できるドキュメントの作成に利用されます。HTML は国際的な標準規格であり、その仕様は World Wide Web Consortium によって保守されています。HTML の現行のバージョンは HTML 4.01 ですが、HTML5 仕様が策定段階にあります。

    HTML についてのドキュメント

    HTML 概論
    構文に基き、ページに基本的な情報とセマンティクスを提供します。
    HTML 要素リファレンス
    異なるブラウザによってサポートされている要素についての詳細を知る。
    HTML 属性リスト
    要素に関連付けられているものを含む、すべての属性の一覧を見る。
    HTML5
    新たな HTML5 API と要素について、 サポート状況と合わせて学ぶ。
    HTML フォーム構築ガイド
    HTML に於いて、フォームはもっとも複雑なパーツです。このガイドは、フォームの構造、スタイリング、ブラウザのサポート、カスタムコントロールをマスターするのに役立ちます。
    コピー & ペーストの弊害
    Web の技術は他のページのソースを閲覧したり、コピー & ペーストによって学ばれる面があります。しかし、これはバッドノウハウが残り続けてしまう面も併せ持っています。
    Canvas を利用してグラフィックを描画する
    プログラマブルな画像処理を可能にする新たな要素についてです。<canvas> はグラフやUI要素、その他の画像処理のクライアント上での描画を可能にします。
    速く読み込まれる HTML ページを作成するための秘訣
    最適化された Web ページは、訪問者により反応の良いサイトを提供するだけでなく、Web サービスやインターネットへの接続負荷を軽減します。
