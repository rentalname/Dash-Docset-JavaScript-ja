    以下のリストは、HTML5 で導入された新しいものを含む、すべての HTML 要素リファレンスです。

    HTML 要素 は、HTML ドキュメントがどのように構築されるか、HTML ドキュメントのどの場所にどのコンテンツが置かれるべきかを指定するコンポーネントです。要素の名前は角括弧で囲まれ (<要素名>)、「タグ」と呼ばれます。要素が他のコンテンツを含む場合、同じ要素名の頭にスラッシュが付く「閉じタグ」(</要素名>) で終わります。画像要素など、一部のタグは閉じる必要がありません。これらは「空要素」として知られています。HTML ドキュメントはこれらの要素のツリーを含みます。各要素には、それを表す名前がつけられています。例えば、<title> 要素は、ドキュメントのタイトルを表します。以下は、HTML 要素のアルファベット順のリストです。

    HTML 要素

    A

    • <a> (アンカーまたはハイパーリンク)
    • <abbr> (略語)
    • <acronym> (頭字語) 廃止 HTML5 — 代わりに <abbr> を使用すること
    • <address> (アドレスまたは連絡先情報)
    • <applet> 非推奨 HTML4 廃止 HTML5 — 代わりに <object> を使用すること
    • <area> (イメージマップのハイパーリンク)
    • <article> (独立した記事) HTML5
    • <aside> (メインではないコンテンツ) HTML5
    • <audio> (埋め込みオーディオストリーム) HTML5

    B

    • <b> (太字)
    • <base> (基準 URL)
    • <basefont> (基準フォント) 非推奨 HTML4 廃止 HTML5
    • <bdi> (双方向アイソレート) HTML5
    • <bdo> (双方向オーバーライド)
    • <bgsound> (バックグラウンド・サウンド)
    • <big> 廃止 HTML5
    • <blink> (点滅)
    • <blockquote> (段落引用)
    • <body> (HTML 本体)
    • <br> (改行)
    • <button> (対話型ボタン)

    C

    • <canvas> (動的グラフィック描画領域) HTML5
    • <caption> (表題)
    • <center> 非推奨 HTML4廃止 HTML5
    • <cite> (引用元)
    • <code> (コード片)
    • <col> (テーブル列)
    • <colgroup> (テーブル列グループ)
    • <command> (ユーザ操作制御) HTML5

    D

    • <datalist> (予め定義されたオプションのリスト) HTML5
    • <dd> (定義記述)
    • <del> (削除テキスト)
    • <details> (オンデマンド制御) HTML5
    • <dfn> (定義)
    • <dir> (ディレクトリリスト) 非推奨 HTML4 廃止 HTML5
    • <div> (ドキュメント区分マーク)
    • <dl> (定義リスト)
    • <dt> (定義用語)

    E

    • <em> (強調)
    • <embed> (埋め込み) HTML5

    F

    • <fieldset> (フォーム制御のセット)
    • <figcaption> (図表キャプション) HTML5
    • <figure> (図表コンテナ) HTML5
    • <font> (フォント) 非推奨 HTML4 廃止 HTML5
    • <footer> (フッタコンテナ) HTML5
    • <form> (ユーザが送信可能なフォーム)
    • <frame> (フレーム) 廃止 HTML5
    • <frameset> (フレームセット) 廃止 HTML5

    G

    H

    • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> (見出し)
    • <head> (ドキュメントのメタデータコンテナ)
    • <header> (見出しコンテナ) HTML5
    • <hgroup> (見出しグループ) HTML5
    • <hr> (主題の区切り、水平線)
    • <html> (ドキュメントのルート)

    I

    • <i> (オフセットテキスト、イタリック体)
    • <iframe> (インラインフレーム)
    • <img> (画像)
    • <input> (入力フォーム制御)
    • <ins> (挿入文)
    • <isindex> 非推奨 HTML4

    J

    K

    • <kbd> (キーボード入力)
    • <keygen> (キーペア生成) HTML5

    L

    • <label> (フォーム制御ラベル)
    • <legend> (フィールドセットの説明)
    • <li> (リスト項目)
    • <link> (ドキュメント内部に関連付けられたメタデータ)
    • <listing> (コードリスト) 非推奨 HTML3.2

    M

    • <map> (イメージマップ)
    • <mark> (マーク付きテキスト) HTML5
    • <marquee>
    • <menu> (メニューリスト) HTML4 で非推奨だったが HTML5 で復活した
    • <meta> (メタデータ)
    • <meter> (スカラーゲージ) HTML5

    N

    • <nav> (ナビゲーションコンテナ) HTML5
    • <nobr> (改行禁止)
    • <noframes> (フレームなし) 廃止 HTML4
    • <noscript> (スクリプト使用不可の場合のフォールバックコンテンツ)

    O

    • <object> (埋め込みオブジェクト)
    • <ol> (順序付きリスト)
    • <optgroup> (オプショングループ)
    • <option> (オプション項目)
    • <output> (計算結果) HTML5

    P

    • <p> (段落)
    • <param> (オブジェクトパラメータ)
    • <plaintext> (プレーンテキスト) 非推奨 HTML2
    • <pre> (Preformatted Text)
    • <progress> (進捗インジケータ) HTML5

    Q

    • <q> (引用)

    R

    (Firefox で Ruby 要素を表示するには、https://addons.mozilla.org/ja/firefox/addon/1935 や https://addons.mozilla.org/ja/firefox/addon/6812 などのアドオンをインストールする必要があります。)

    • <rp> (ルビの括弧書き) HTML5未実装
    • <rt> (ルビのテキスト) HTML5未実装
    • <ruby> (ルビ) HTML5未実装 (参照 バグ 33339)

    S

    • <s> (取り消し線) 非推奨 HTML4
    • <samp> (サンプルテキスト)
    • <script> (埋め込みスクリプト)
    • <section> (ドキュメントのアウトラインセクション) HTML5
    • <select> (選択リスト)
    • <small> (小型テキスト)
    • <source> (埋め込みメディア) HTML5
    • <spacer> (レイアウト用スペース)
    • <span> (インラインテキスト範囲)
    • <strike> (テキスト取り消し) 非推奨 HTML4
    • <strong> (強い強調)
    • <style> (CSS スタイル情報)
    • <sub> (下付き)
    • <summary> (詳細の要約) HTML5
    • <sup> (上付き)

    T

    • <table> (テーブル)
    • <tbody> (テーブル本体)
    • <td> (テーブルデータのセル)
    • <textarea> (複数行入力フォーム)
    • <tfoot> (テーブルフッタ)
    • <th> (テーブル見出し)
    • <thead> (テーブルヘッダ)
    • <time> (日付または時刻) HTML5
    • <title> (HTML ドキュメントタイトル)
    • <tr> (テーブル行)
    • <track>HTML5
    • <tt> (テレタイプテキスト) 非推奨 HTML5

    U

    • <u> (HTML4 までは下線、HTML5 では曖昧) 非推奨 HTML4 HTML5
    • <ul> (順序なしリスト)

    V

    • <var> (変数テキスト)
    • <video> (埋め込みビデオストリーム) HTML5

    W

    • <wbr> (改行許可) HTML5

    X

    • <xmp> (例示) 非推奨 HTML3.2 廃止 HTML4

    Y

    Z
