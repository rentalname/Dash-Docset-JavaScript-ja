    本ページでは、Firefox 内蔵の開発ツールで使用している全キーボードショートカットを一覧化しています。

    最初のセクションではそれぞれのツールを開くためのショートカットを、次のセクションではツールボックス自体に適用するショートカットを一覧化します。その後に各ツールのセクションがあり、そこではツール内で使用できるショートカットを一覧化しています。

    アクセスキーはロケールに依存するため、本ページでは文書化していません。

    ツールの開始と終了

    これらのショートカットは、メインのブラウザウィンドウで特定のツールを開始します。ツールボックスに含まれているツールは、アクティブであるときに同じショートカットで終了します。ブラウザコンソールのように新しいウィンドウを開くツールでは、ツールを終了するためにウィンドウを閉じなければなりません。

      Windows Mac OS X Linux
    ツールボックス (最後にアクティブであったツールも開きます) Ctrl + Shift + I Cmd + Opt + I Ctrl + Shift + I
    Web コンソール1 Ctrl + Shift + K Cmd + Opt + K Ctrl + Shift + K
    インスペクタ Ctrl + Shift + C Cmd + Opt + C Ctrl + Shift + C
    デバッガ Ctrl + Shift + S Cmd + Opt + S Ctrl + Shift + S
    スタイルエディタ Shift + F7 Shift + F71 Shift + F7
    プロファイラ Shift + F5 Shift + F51 Shift + F5
    ネットワークモニタ Ctrl + Shift + Q Cmd + Opt + Q Ctrl + Shift + Q
    開発ツールバー (オン・オフの切り替え) Shift + F2 Shift + F21 Shift + F2
    レスポンシブデザインビュー (オン・オフの切り替え) Ctrl + Shift + M Cmd + Opt + M Ctrl + Shift + M
    ブラウザコンソール Ctrl + Shift + J Cmd + Shift + J Ctrl + Shift + J
    スクラッチパッド Shift + F4 Shift + F4 Shift + F4

    1. 他のツールボックス内ツールと異なり、このショートカットでは Web コンソールが閉じません。代わりに、Web コンソールのコマンドラインにフォーカスがあたります。Web コンソールを閉じるには、ツールボックスのグローバルなショートカットである Ctrl+Shift+I (Mac では Cmd+Opt+I) を使用してください。

    ツールボックス

    これらのショートカットはどのツールがアクティブであるかにかかわらず、ツールボックスを開いているときに動作します。

      Windows Mac OS X Linux
    開発ツールを左から右に切り替える Ctrl + ] Cmd + ] Ctrl + ]
    開発ツールを右から左に切り替える Ctrl + [ Cmd + [ Ctrl + [
    開発ツールのオプションを開く Ctrl + Shift + O Cmd + Shift + O Ctrl + Shift + O

    これらのショートカットは、ツールボックス内の全ツールで動作します。

      Windows Mac OS X Linux
    フォントサイズを拡大する Ctrl + + Cmd + + Ctrl + +
    フォントサイズを縮小する Ctrl + - Cmd + - Ctrl + -
    フォントサイズのリセット Ctrl + 0 Cmd + 0 Ctrl + 0

    ソースエディタ

    以下の表は、ソースエディタのデフォルトのショートカットの一覧です。

    Vim や Emacs のキーバインドも選択できます。これらを選択するには、about:config で設定項目 devtools.editor.keymap を選択して、"vim" または "emacs" を設定してください。この設定により、ソースエディタを使用するすべての開発ツールで選択したキーバインドを使用します。設定を反映させるには、エディタを開き直さなければなりません。

    Firefox 31 より、devtools.editor.keymap に "sublime" を設定することにより Sublime Text のキーバインドを選択できます。

      Windows Mac OS X Linux
    指定行へ移動 Ctrl + J Cmd + J Ctrl + J
    ファイル内を検索 Ctrl + F Cmd + F Ctrl + F
    再検索 Ctrl + G Cmd + G Ctrl + G
    すべて選択 Ctrl + A Cmd + A Ctrl + A
    切り取り Ctrl + X Cmd + X Ctrl + X
    コピー Ctrl + C Cmd + C Ctrl + C
    貼り付け Ctrl + V Cmd + V Ctrl + V
    元に戻す Ctrl + Z Cmd + Z Ctrl + Z
    やり直し Ctrl + Shift + Z / Ctrl + Y Cmd + Shift + Z / Cmd + Y Ctrl + Shift + Z / Ctrl + Y
    インデント挿入 Tab Tab Tab
    インデント削除 Shift + Tab Shift + Tab Shift + Tab
    行を上へ移動 Alt + ↑ Alt + ↑ Alt + ↑
    行を下へ移動 Alt + ↓ Alt + ↓ Alt + ↓
    行のコメント化/コメント解除 Ctrl + / Cmd + / Ctrl + /

    1. 一部の Mac はデフォルトで、ファンクションキーが特別な機能の使用に再割り当てされています: 例えば、画面の明るさや音量の変更です。これらのキーを標準的なファンクションキーとして使用するためのガイド (日本語版) をご覧ください。再割り当てされたキーを標準的なファンクションキーとして使用するには、同時に Function キーも押下してください (よってプロファイラを開くには、Shift + Function + F5 を押してください)。
