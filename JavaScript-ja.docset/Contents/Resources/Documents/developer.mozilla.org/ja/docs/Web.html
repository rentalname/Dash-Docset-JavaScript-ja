    開発者向けのWeb技術

    オープンWebは信じられないほどの機会を開発者に提供します。これらの技術を最大限に活用するには、それらを使用する方法を知っておく必要があります。以下に私たちのWeb技術のドキュメントへのリンクがあります。

    Web開発者向けのドキュメント

    Web開発リファレンス
    HTMLやCSSを含むWeb技術のリファレンスリストです。
    Web開発者ガイド
    Web開発ガイドは、実際にあなたのしたいことや必要のあることにWeb技術を使って実現するための有用な手引を提供します。
    Web開発者のためのチュートリアル
    このチュートリアルのリストはステップ・バイ・ステップでAPI、技術などの幅広いトピック領域を学習できます。

    View All...

    Web技術リファレンス

    Web APIs
    Reference material for each of the interfaces that comprise the Web's APIs, including the DOM and all of the related APIs and interfaces you can use to build Web content and apps.
    HTML
    HyperText Markup Language (HTML)はWebコンテンツを記述し定義するのに使う言語です。
    CSS
    Cascading Style Sheets(CSS)はWebコンテンツの外観を記述するのに使う言語です。
    SVG
    Scalable Vector Graphics(SVG)は大きさに関係なくスムーズに拡大縮小できる、ベクトルや図形の集合体としての画像の説明をします。
    MathML
    Mathematical Markup(MathML)言語により複雑な数学の方程式や構文を表示することができます。
